7.1.2 了解类型

打印变量的类型

```golang
year := 2022
fmt.Printf("Type %T for %v\n", year, year)
// Type int for 2022

days := 365.2425
fmt.Printf("Type %T for %[1]v\n", days, days)
// Type float64 for 365.2425
```
